# Changelog

## 2026-04-13 v3 — 可玩性大爆發第二彈

### 新植物
- 🌵 **仙人掌 (Cactus)** — 穿透射擊，可打到地下礦工殭屍，命中後繼續穿透下一個目標

### 新殭屍
- 👺 **小鬼殭屍 (Imp)** — 巨人殭屍死亡時從背上跳下，速度極快

### 新系統
- ⭐ **植物升級** — 點擊已種植物或右鍵格位花費陽光升級（★標記），每棵植物有獨特升級效果：
  - 豌豆射手：射速+30%
  - 向日葵：產量+10、間隔縮短
  - 堅果牆：HP+120
  - 雙發射手：三連發
  - 冰豆射手：緩速+2s
  - 稜鏡花：折射傷害+5
  - 賭徒花：暴擊率+15%
  - 大嘴花：咀嚼時間-2s
  - 火炬樹：濺射範圍+50%
  - 仙人掌：穿透+1目標
- ✨ **Power-up 掉落** — 殭屍死亡 12% 機率掉落：
  - ✨ 陽光爆發：+50 陽光
  - ⏱️ 急速冷卻：所有冷卻歸零
  - 💚 全體治療：所有植物回 30% HP
  - 🛡️ 植物護盾：3 秒無敵
- ⏩ **速度控制** — 1x / 1.5x / 2x 切換
- 🔴 **行危險指示器** — 每排殭屍≥3時黃色警告，≥5時紅色危險
- 🟢 **格位高亮** — 選植物時顯示可放置（綠）/ 不可放置（紅）
- 🛡️ **護盾視覺** — Power-up 護盾生效時棋盤邊框藍光

### 新成就
- 強化新手：升級 3 棵植物
- 拾荒者：拾取 5 個 Power-up

### 新音效
- upgrade / powerup

### 改進
- 點擊已種植物直接觸發升級
- 升級後植物顯示★星號
- 升級植物有金黃 box-shadow
- Power-up 浮動動畫
- 仙人掌豌豆綠色大粒子
- 小鬼殭屍紫色主題
- 遊戲結束顯示升級次數
